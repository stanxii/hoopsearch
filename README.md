# hoopsearch
一个成长中的小搜索引擎

#### 
打算为[hoopics](http://www.hoopics.cn)实现一个搜索引擎, 满足图片搜索, 用户搜索及模糊查询等相关功能.  
利用业余时间边研究边写代码, 进度有点儿感人..., 如果有对搜索感兴趣童鞋一起写呀:)
